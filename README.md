## 關於這個 Repo

這個 Repository 是本場活動的現場示範材料，展示 **AI 如何實際參與 Agile 工作流程**——從需求拆解、Story 精煉、估算，到 Sprint 執行，全程以 [Google Antigravity](https://antigravity.google/?hl=zh-tw) + Skills 示範。

## 活動主題

當 AI 真正加入敏捷團隊，站立會議、需求拆解、估算、回顧會議、甚至決策本身，都會長得不一樣。

本次活動從**工程師視角**探討：

- AI 對 Agile / Scrum 流程的實際影響
- AI 在敏捷工作流中的角色可視化
- 人與 AI 的協作模式（而非取代關係）
- 可帶回團隊討論的實用框架
